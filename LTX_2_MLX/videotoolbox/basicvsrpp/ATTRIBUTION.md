# Attribution

`videotoolbox/basicvsrpp/` is an independent MLX reimplementation of BasicVSR++, written
from the published architecture as a spec -- no upstream code is bundled. The model weights
are the upstream project's, redistributed under its license (see `weights/README.md` for
the downloads).

## BasicVSR++

Video Super-Resolution with Enhanced Propagation and Alignment" (CVPR 2022). The
implementation and pretrained weights are from OpenMMLab's MMagic (formerly MMEditing).
